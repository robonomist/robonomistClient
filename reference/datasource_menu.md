# Structured menu of available datasources and data tables

Structured menu of available datasources and data tables

## Usage

``` r
datasource_menu(datasource = NULL, lang = NULL)
```

## Arguments

- datasource, :

  Character, Datasource name

- lang, :

  Two-letter language code, e.g. "en" or "sv".

## Value

Nested list of datasource_menu and datasource_menu_item objects
